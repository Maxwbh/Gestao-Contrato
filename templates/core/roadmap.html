Fase 1 — Interface & Design

Modernização visual e experiência do usuário

Em progresso
Troca de tema: Bootstrap → Materialize CSS
Próximo

Substituir Bootstrap 5 pelo Materialize CSS (Material Design do Google). Proporciona elevação de cards, paleta de cores dinâmica, ripple em botões, navegação lateral (sidenav) e componentes nativos como chips, FAB e datepickers.

Materialize CSS 1.x Material Design UI/UX Alto impacto
Cards reposicionáveis e redimensionáveis
Concluído

Cards arrastáveis com SortableJS na lista de contratos. Tamanhos configuráveis por card (2, 3, 6, 9, 12 colunas), tamanho mínimo por tipo de conteúdo e layout persistido no localStorage.

SortableJS localStorage Concluído
Modo escuro (Dark Mode)
Planejado

Alternância entre tema claro e escuro, respeitando a preferência do sistema (prefers-color-scheme) e com persistência na preferência do usuário.

CSS Variables localStorage

Fase 2 — AG Grid

Grade de dados avançada para todas as listagens

Próximo
Integração AG Grid Community
Próximo

Substituir todas as tabelas HTML estáticas pelo AG Grid Community (versão gratuita). Grade de alto desempenho com virtualização de linhas, ideal para grandes volumes de dados.

AG Grid 31+ JavaScript REST API
Filtros por coluna nativos
Próximo

Cada coluna terá filtro embutido (texto, número, data, seleção). Filtros combinados aplicados via API com debounce para não sobrecarregar o servidor.

Column Filters Debounce Server-side
Busca global por listagem
Próximo

Campo de busca rápida sobre todas as colunas visíveis da grade. Realce de termos encontrados, busca case-insensitive e suporte a múltiplos termos separados por espaço.

Quick Filter Highlight
Paginação configurável (5, 10, 15, 25…100)
Em progresso

Todas as listagens com seletor de registros por página: 5, 10, 15, 25, 50 e 100. AG Grid fornece paginação client-side e server-side com API de blocos infinitos (infinite row model).

Pagination Per-page selector Em progresso
Ordenação multi-coluna
Planejado

Ordenar por múltiplas colunas simultaneamente (shift+clique). Indicador visual da ordem de prioridade. Persistência da ordenação preferida no localStorage por listagem.

Multi-sort localStorage
Exportação CSV / Excel
Planejado

Exportar a grade atual (com filtros aplicados) para CSV ou Excel diretamente via AG Grid. Sem necessidade de bibliotecas extras — exportação nativa da Community Edition.

CSV Export Excel Export
