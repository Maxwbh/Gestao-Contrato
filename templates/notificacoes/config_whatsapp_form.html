Guia de Configuração por Provedor
Evolution API (self-hosted)
  • API URL: http://servidor:8080
  • API Key: chave definida em AUTHENTICATION_API_KEY
  • Instância: nome criado no painel Evolution
  • Escaneie o QR Code na instância para conectar
Z-API
  • API URL: https://api.z-api.io
  • API Key: token da instância Z-API
  • Instância: instance ID da Z-API
  • Client-Token: cabeçalho obrigatório da Z-API
Twilio
  • Account SID + Auth Token do console Twilio
  • Número: whatsapp:+5511999999999
  • Habilite WhatsApp Sandbox no console
Meta (WhatsApp Business API)
  • Account SID: ID da conta Meta Business
  • API Key: token de acesso permanente
  • API URL: endpoint da Cloud API Meta
  • Número: número verificado no Meta Business
