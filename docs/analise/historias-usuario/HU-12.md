# HU-12 — Calcular Cessão de Direitos

## História do Usuário

**Como** operador da imobiliária  
**Quero** calcular a taxa de cessão de direitos de um contrato  
**Para** informar ao comprador o custo de transferência do contrato para terceiros antes de formalizar a operação

---

## Pré-condições

- Contrato com `status = ATIVO`
- `percentual_cessao` configurado no contrato (padrão: 3%)
- Usuário com permissão de visualizar cálculo de cessão

---

## Critérios de Aceitação

1. `GET /contratos/<id>/cessao/` exibe formulário com campo **data de referência** (opcional, padrão: hoje)
2. `POST /contratos/<id>/cessao/` executa `Contrato.calcular_cessao(data_cessao)` e exibe resultado
3. Fórmula de cálculo:
   ```
   saldo_devedor = calcular_saldo_devedor()
   taxa_cessao = saldo_devedor × percentual_cessao / 100
   ```
4. Resultado exibe:
   - Saldo devedor na data de referência
   - Percentual de cessão (%)
   - Valor da taxa de cessão (R$)
5. O cálculo é **apenas informativo** — não gera cobrança automática nem altera o status do contrato
6. O `percentual_cessao` é configurável por contrato em "Editar Contrato"

---

## Regras de Negócio

- RN-01: **PRICE**: `saldo_devedor = SUM(valor_atual)` de parcelas NORMAL não pagas (PMTs futuros comprometidos)
- RN-02: **SAC**: `saldo_devedor = SUM(amortizacao)` de parcelas NORMAL não pagas (principal restante real); fallback para SUM(valor_atual) se amortizacao não preenchida
- RN-03: `percentual_cessao` padrão = 3%; configurável por contrato (0–100%)
- RN-04: Cálculo não altera status, parcelas ou qualquer campo do contrato

---

## Definição de Pronto

- [ ] Formulário com campo de data opcional
- [ ] Resultado exibe saldo devedor, percentual e valor da taxa
- [ ] Cálculo correto para amortização PRICE e SAC
- [ ] Percentual configurável via edição do contrato

---

## Cenários de Teste

### CT-01 — Cessão PRICE
**Dado** contrato PRICE com 60 parcelas restantes de R$ 1.000 cada  
**E** `saldo_devedor = SUM(valor_atual) = R$ 60.000`, `percentual_cessao = 3%`  
**Quando** calcula cessão  
**Então** `taxa_cessao = 60.000 × 3% = R$ 1.800`

### CT-02 — Cessão SAC
**Dado** contrato SAC com `SUM(amortizacao) = R$ 45.000`, `percentual_cessao = 3%`  
**Quando** calcula cessão  
**Então** `taxa_cessao = 45.000 × 3% = R$ 1.350`

### CT-03 — Data de referência futura
**Dado** data = 6 meses no futuro (contrato com 6 parcelas pagas no intervalo)  
**Quando** calcula cessão  
**Então** saldo calculado considerando parcelas restantes após a data informada (parcelas com `data_vencimento > data_referencia`)

### CT-04 — Percentual personalizado
**Dado** `percentual_cessao = 5%` configurado no contrato  
**Quando** calcula  
**Então** `taxa_cessao = saldo × 5%`

---

## Fluxo do Processo

```
Operador acessa Cessão do Contrato
    │
    ▼
GET /contratos/<id>/cessao/
    │   Formulário: campo data_referencia (opcional, padrão: hoje)
    │   Informação: percentual_cessao atual do contrato
    │
    ▼
POST /contratos/<id>/cessao/
    │   data_cessao = data informada ou date.today()
    │
    ├── contrato.calcular_cessao(data_cessao):
    │
    │   saldo_devedor = calcular_saldo_devedor()
    │       PRICE → SUM(valor_atual) parcelas NORMAL não pagas
    │       SAC   → SUM(amortizacao) parcelas NORMAL não pagas
    │
    │   taxa_cessao = saldo_devedor × percentual_cessao / 100
    │
    └── Retorna {saldo_devedor, percentual_cessao, taxa_cessao}
    │
    ▼
Renderiza resultado:
    ├── Saldo Devedor: R$ X.XXX,XX
    ├── Percentual de Cessão: X,XX%
    └── Taxa de Cessão: R$ XXX,XX
    │
    ▼
Cálculo exibido (apenas informativo — nenhum dado alterado) ✅
```
