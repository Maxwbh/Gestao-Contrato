# HU-09 — Quitação Manual / Simulador de Antecipação

## História do Usuário

**Como** operador da imobiliária  
**Quero** simular e registrar a quitação antecipada de parcelas com desconto negociado  
**Para** atender compradores que desejam quitar total ou parcialmente o contrato antes do prazo, com geração de recibo em PDF

---

## Pré-condições

- Contrato `ATIVO`
- Pelo menos uma parcela `NORMAL` com `pago = False`
- Usuário com permissão de quitação

---

## Critérios de Aceitação

1. `GET /financeiro/contrato/<id>/simulador/` exibe formulário com parcelas NORMAL não pagas selecionáveis e campo de desconto (% ou valor fixo)
2. `POST action=preview` retorna prévia com: parcelas selecionadas, desconto aplicado, valor final por parcela, total — **sem persistir**
3. `POST action=aplicar` persiste os pagamentos das parcelas selecionadas com desconto
4. Para cada parcela quitada:
   - Cria `HistoricoPagamento` com `origem_pagamento = MANUAL` e `antecipado = True`
   - Aplica o desconto sobre `valor_atual`
   - Marca `pago = True` e `status_boleto = PAGO`
5. PDF de recibo de quitação gerado via `recibo_service` e disponível em `/contrato/<id>/recibo-antecipacao.pdf`
6. Se todas as parcelas NORMAL forem quitadas, `contrato.status = QUITADO`
7. A quitação manual **não** é bloqueada por reajuste pendente

---

## Regras de Negócio

- RN-01: Desconto pode ser percentual ou valor fixo por parcela, mas nunca superior a `valor_atual` da parcela
- RN-02: Apenas parcelas `NORMAL` entram na simulação; intermediárias são tratadas separadamente
- RN-03: Preview (`action=preview`) nunca persiste dados — é somente simulação
- RN-04: O recibo PDF deve conter: identificação das partes, parcelas quitadas, desconto, valor pago, data e espaço para assinaturas
- RN-05: Quitação ignora `bloqueio_boleto_reajuste` — apenas emissão de boletos é afetada pelo bloqueio

---

## Definição de Pronto

- [ ] `HistoricoPagamento` criado com `antecipado = True` para cada parcela
- [ ] Desconto aplicado e valor_pago registrado corretamente
- [ ] Recibo PDF disponível para download imediato
- [ ] Contrato `QUITADO` quando todas as NORMAL são quitadas
- [ ] Preview funciona sem persistir dados
- [ ] Desconto inválido rejeitado com mensagem

---

## Cenários de Teste

### CT-01 — Antecipar 5 parcelas com desconto de 10%
**Dado** 5 parcelas NORMAL selecionadas, desconto = 10%  
**Quando** `POST action=preview`  
**Então** retorna: `valor_final = valor_atual × 0,90` para cada, `total = SUM(valores_finais)`; nada persistido

### CT-02 — Aplicar antecipação
**Dado** preview confirmado  
**Quando** `POST action=aplicar`  
**Então** 5 `HistoricoPagamento` criados, 5 parcelas com `pago = True`, PDF de recibo gerado

### CT-03 — Quitar todas as parcelas do contrato
**Dado** operador seleciona todas as parcelas NORMAL restantes  
**Quando** aplica  
**Então** `contrato.status = QUITADO` após a operação

### CT-04 — Desconto maior que o valor da parcela
**Dado** desconto = R$ 2.000 em parcela de R$ 1.500  
**Quando** submete formulário  
**Então** rejeita com mensagem "Desconto não pode superar o valor da parcela"

### CT-05 — Sem parcelas elegíveis
**Dado** contrato com todas as parcelas pagas  
**Quando** acessa o simulador  
**Então** exibe mensagem "Não há parcelas em aberto para antecipação"

## Fluxo do Processo

```
Operador acessa simulador
    │
    ▼
GET /financeiro/contrato/<id>/simulador/
    │   Lista parcelas NORMAL não pagas com checkbox
    │   Campo: desconto (% ou R$)
    │
    ▼
POST action=preview
    │
    ├── Para cada parcela selecionada:
    │       valor_final = valor_atual − desconto_calculado
    │       Validar: valor_final ≥ 0
    │
    └── Retorna tabela de preview:
            [{parcela, valor_atual, desconto, valor_final}]
            total_a_pagar = SUM(valor_final)
    │
    (Operador revisa e confirma)
    │
    ▼
POST action=aplicar
    │
    ├── Para cada parcela selecionada:
    │       HistoricoPagamento.create(
    │           parcela=parcela,
    │           valor_pago=valor_final,
    │           data_pagamento=hoje,
    │           origem_pagamento=MANUAL,
    │           antecipado=True
    │       )
    │       parcela.pago = True
    │       parcela.status_boleto = PAGO
    │       parcela.save()
    │
    ├── recibo_service.gerar_recibo(parcelas_quitadas, desconto, usuario)
    │       → PDF com dados das partes, parcelas, desconto, assinatura
    │
    └── Verificar quitação do contrato:
            Se todas NORMAL pagas → contrato.status = QUITADO
    │
    ▼
GET /contrato/<id>/recibo-antecipacao.pdf
    │   Download do PDF de recibo
    │
    ▼
Quitação registrada ✅
```
