# HU-14 — Gestão de Prestações Intermediárias

## História do Usuário

**Como** operador da imobiliária  
**Quero** gerenciar as prestações intermediárias de um contrato após sua criação — criar, editar, excluir, registrar pagamento e gerar boleto individualmente  
**Para** ajustar o cronograma de intermediárias conforme negociações pós-contrato sem precisar recriar o contrato

---

## Pré-condições

- Contrato com `status = ATIVO`
- Usuário com permissão de gerenciar intermediárias

---

## Critérios de Aceitação

1. `GET /contratos/<contrato_id>/intermediarias/` → lista todas as `PrestacaoIntermediaria` do contrato com status de pagamento
2. `GET /contratos/intermediarias/<pk>/` → detalhe de uma intermediária
3. `POST /contratos/<contrato_id>/intermediarias/criar/` → cria nova intermediária (retorna JSON)
   - Campos: `data_vencimento`, `valor`, `descricao` (opcional)
   - Validação: soma total das intermediárias ≤ 80% do valor financiado
4. `POST /contratos/intermediarias/<pk>/atualizar/` → edita data e valor de uma intermediária não paga (retorna JSON)
5. `POST /contratos/intermediarias/<pk>/excluir/` → exclui intermediária não paga (retorna JSON)
6. `POST /contratos/intermediarias/<pk>/pagar/` → registra pagamento da intermediária (retorna JSON)
   - Campos: `valor_pago`, `data_pagamento`, `forma_pagamento`
   - Marca `pago = True` e registra `HistoricoPagamento`
7. `POST /contratos/intermediarias/<pk>/gerar-boleto/` → gera boleto para a intermediária (retorna JSON)
   - Usa conta bancária ativa da imobiliária
   - Aplica mesmas validações do boleto normal (HU-03)
8. `GET /contratos/<contrato_id>/intermediarias/api/` → lista JSON das intermediárias (para painéis dinâmicos)

---

## Regras de Negócio

- RN-01: Intermediária paga não pode ser editada ou excluída
- RN-02: Ao criar nova intermediária, a soma total não pode ultrapassar 80% do valor financiado
- RN-03: O pagamento cria `HistoricoPagamento` com `origem_pagamento = MANUAL` e `tipo_parcela = INTERMEDIARIA`
- RN-04: A geração de boleto para intermediária obedece ao mesmo bloqueio de reajuste das parcelas normais (HU-06)
- RN-05: Excluir intermediária não atualiza automaticamente o PMT das parcelas normais — operação é manual

---

## Definição de Pronto

- [ ] CRUD completo via endpoints JSON (criar, atualizar, excluir)
- [ ] Pagamento cria `HistoricoPagamento` com tipo correto
- [ ] Validação de 80% aplicada ao criar
- [ ] Intermediária paga bloqueada para edição/exclusão
- [ ] Boleto gerado obedece ao bloqueio de reajuste

---

## Cenários de Teste

### CT-01 — Criar intermediária válida
**Dado** contrato com valor financiado R$ 100.000 e soma atual de intermediárias = R$ 40.000  
**Quando** cria nova intermediária de R$ 30.000  
**Então** criada com sucesso (soma = R$ 70.000 = 70%, dentro do limite)

### CT-02 — Criar intermediária excede 80%
**Dado** soma atual = R$ 70.000  
**Quando** tenta criar intermediária de R$ 15.000 (soma ficaria R$ 85.000 = 85%)  
**Então** rejeita com mensagem de validação

### CT-03 — Editar intermediária não paga
**Dado** intermediária com `pago = False`  
**Quando** atualiza `data_vencimento` e `valor`  
**Então** atualizada com sucesso

### CT-04 — Editar intermediária paga — bloqueada
**Dado** intermediária com `pago = True`  
**Quando** tenta editar  
**Então** rejeita com mensagem "Intermediária já paga não pode ser editada"

### CT-05 — Pagar intermediária
**Dado** intermediária não paga  
**Quando** registra pagamento com `valor_pago`, `data_pagamento = hoje`, `forma_pagamento = PIX`  
**Então** `pago = True`, `HistoricoPagamento` criado com `tipo_parcela = INTERMEDIARIA`

---

## Fluxo do Processo

```
Operador acessa intermediárias do contrato
    │
    ▼
GET /contratos/<id>/intermediarias/
    │   Lista: data, valor, status, ações disponíveis
    │
    ├──► Criar nova:
    │       POST /contratos/<id>/intermediarias/criar/
    │           ├── Validar soma ≤ 80% do financiado
    │           └── Criar PrestacaoIntermediaria
    │
    ├──► Editar:
    │       POST /contratos/intermediarias/<pk>/atualizar/
    │           └── Verificar pago=False antes de editar
    │
    ├──► Pagar:
    │       POST /contratos/intermediarias/<pk>/pagar/
    │           ├── HistoricoPagamento.create(tipo=INTERMEDIARIA, origem=MANUAL)
    │           └── intermediaria.pago = True
    │
    ├──► Gerar Boleto:
    │       POST /contratos/intermediarias/<pk>/gerar-boleto/
    │           ├── Verificar pode_gerar_boleto() (bloqueio reajuste)
    │           └── BoletoService.gerar_boleto(intermediaria, conta_bancaria)
    │
    └──► Excluir:
            POST /contratos/intermediarias/<pk>/excluir/
                └── Verificar pago=False antes de excluir
```
