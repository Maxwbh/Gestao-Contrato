# HU-11 — Calcular Rescisão Contratual

## História do Usuário

**Como** operador da imobiliária  
**Quero** calcular o valor a devolver ao comprador em caso de rescisão do contrato  
**Para** realizar o processo de rescisão de forma transparente, com breakdown completo das retenções conforme as cláusulas contratuais

---

## Pré-condições

- Contrato com `status = ATIVO`
- Percentuais de rescisão configurados: `percentual_fruicao`, `percentual_multa_rescisao_penal`, `percentual_multa_rescisao_adm`
- Usuário com permissão de visualizar cálculo de rescisão

---

## Critérios de Aceitação

1. `GET /contratos/<id>/rescisao/` exibe formulário com campo para **data da rescisão**
2. `POST /contratos/<id>/rescisao/` executa `Contrato.calcular_rescisao(data_rescisao)` e exibe resultado detalhado
3. O botão "Calcular" **não exibe diálogo de confirmação** (classe CSS `no-confirm` no botão)
4. O resultado **permanece visível** até nova ação do usuário — não some por timeout automático
5. Breakdown completo exibido:

| Item | Cálculo |
|------|---------|
| Entrada paga | Valor da entrada registrado |
| Parcelas pagas | `SUM(valor_pago)` de parcelas com `pago=True` |
| Intermediárias pagas | Soma das `PrestacaoIntermediaria` pagas |
| **Total pago** | Entrada + parcelas + intermediárias |
| Saldo devedor | `calcular_saldo_devedor()` |
| Meses ocupados | `(data_rescisao.year − data_contrato.year)×12 + (data_rescisao.month − data_contrato.month)` |
| Fruição | `saldo × percentual_fruicao / 100 × meses_ocupados` |
| Multa penal | `saldo × percentual_multa_rescisao_penal / 100` |
| Despesas adm | `saldo × percentual_multa_rescisao_adm / 100` |
| **Total retenções** | Fruição + multa + desp. adm |
| **Valor a devolver** | `max(0, total_pago − total_retenções)` |

6. Se `total_retenções ≥ total_pago` → exibe aviso "As retenções superam o valor pago. Não há valor a devolver." e devolução = R$ 0,00
7. O cálculo é apenas informativo — **não altera o status do contrato automaticamente**
8. Link para edição dos percentuais em "Editar Contrato" exibido no resultado

---

## Regras de Negócio

- RN-01: `devolucao = max(0, total_pago − total_retencoes)` — nunca negativa
- RN-02: Saldo devedor difere por amortização: PRICE = SUM(valor_atual) das NORMAL não pagas; SAC = SUM(amortizacao)
- RN-03: Padrões configuráveis: fruição 0,5%/mês, multa penal 10%, desp. adm 12% — todos sobre o saldo devedor
- RN-04: Percentuais são configurados por contrato, não globalmente
- RN-05: O botão "Calcular" usa classe `no-confirm` para não acionar o diálogo de confirmação de formulários perigosos

---

## Definição de Pronto

- [ ] Resultado exibido com breakdown completo (6 linhas)

## Cenários de Teste

### CT-01 — Rescisão com devolução positiva
**Dado** comprador pagou R$ 50.000 (entrada + 24 parcelas)  
**E** saldo devedor = R$ 100.000, fruição = 0,5%/mês, 24 meses, multa penal = 10%, desp. adm = 12%  
**Quando** calcula rescisão  
**Então**:  
  - fruição = 100.000 × 0,5% × 24 = R$ 12.000  
  - multa penal = 100.000 × 10% = R$ 10.000  
  - desp. adm = 100.000 × 12% = R$ 12.000  
  - total retenções = R$ 34.000  
  - devolução = max(0, 50.000 − 34.000) = R$ 16.000

### CT-02 — Retenções maiores que total pago
**Dado** comprador pagou R$ 20.000 e total de retenções = R$ 30.000  
**Quando** calcula rescisão  
**Então** exibe aviso de sem devolução; `devolucao = R$ 0,00`

### CT-03 — Resultado permanece visível
**Dado** resultado exibido após cálculo  
**Quando** aguarda 10 segundos  
**Então** resultado ainda visível (não foi auto-dismissed)

### CT-04 — Botão sem confirm dialog
**Dado** página de rescisão carregada  
**Quando** operador clica em "Calcular"  
**Então** formulário submetido diretamente sem popup de confirmação

### CT-05 — Diferentes datas de rescisão
**Dado** mesmo contrato  
**Quando** calcula para data 12 meses vs 24 meses  
**Então** fruição dobra proporcionalmente aos meses ocupados

## Fluxo do Processo

```
Operador acessa Rescisão do Contrato
    │
    ▼
GET /contratos/<id>/rescisao/
    │   Formulário: campo data_rescisao (datepicker)
    │   Botão: "Calcular" (class="btn-danger no-confirm")
    │
    ▼
POST /contratos/<id>/rescisao/
    │   data_rescisao = data informada
    │
    ├── contrato.calcular_rescisao(data_rescisao):
    │
    │   saldo_devedor = calcular_saldo_devedor()
    │   meses_ocupados = Δ meses entre data_contrato e data_rescisao
    │
    │   fruicao = saldo × percentual_fruicao / 100 × meses_ocupados
    │   multa_penal = saldo × percentual_multa_rescisao_penal / 100
    │   desp_adm = saldo × percentual_multa_rescisao_adm / 100
    │   total_retencoes = fruicao + multa_penal + desp_adm
    │
    │   total_pago = entrada_paga + parcelas_pagas + intermediarias_pagas
    │   devolucao = max(0, total_pago − total_retencoes)
    │
    └── Retorna dict com todos os campos
    │
    ▼
Renderiza resultado:
    ├── Tabela de valores pagos
    ├── Tabela de retenções (com base no saldo devedor)
    ├── Alert com valor a devolver (verde se > 0, amarelo se = 0)
    └── Link: "Configurar percentuais em Editar Contrato"
    │
    ▼
Resultado permanente na página ✅ (não some automaticamente)
```
